# 2025_SCPC_AI-Challenge

2025 Samsung Collegiate Programming Challenge : 사용자의 일상 사진을 이해하는 on device multi-modal AI 개발


## ⚙️ Execution Environment

### 🖥️ System
* Platform: Google Colab (Linux)
* Python: 3.12.13
* OS: Linux 6.6.113+ (glibc 2.35)

### 🚀 Hardware
* GPU: NVIDIA H100 (Hopper Architecture)
* CUDA: 12.8

### 🧠 Deep Learning Framework
* PyTorch: 2.10.0 (cu128)
* torchvision: 0.25.0 (cu128)

### 🤖 Libraries
* transformers: 4.44.2
